## 请切记要引入相关的注解类
## 当你的action上的路由以 / 开头时，那完整的路由就是它，即不会再将 prefix 添加到它的前面。
## 通常一个完整的路由path等于 Controller的 prefix + Action的 route

- 驼峰法
  - /test/getId
```

<?php declare(strict_types=1);

/**
 * Class TestController
 * @Controller()
 */
class TestController
{
    /**
     * @RequestMapping()
     * @return string
     */
    public function getId()
    {
        return "get id";
    }
}

```
